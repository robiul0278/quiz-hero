What's the difference .forEach(), .map(), .filter() and .find() ?

.forEach(): The .forEach() method executes a provided function once for each array element. It is useful when you want to loop through an array and perform some operation on each element, without changing the original array. It doesn't return anything, it just executes the provided function on each element of the array.


.map(): The .map() method creates a new array with the results of calling a provided function on every element in the calling array. It returns a new array with the same length as the original array, but with the modified elements.


.filter(): The .filter() method creates a new array with all elements that pass the test implemented by the provided function. It returns a new array containing only the elements that meet the condition.


.find(): .find() is another array method in JavaScript that is similar to .filter(), but it returns only the first element in the array that satisfies a provided testing function, rather than returning a new array with all the elements that pass the test.



whats the difference browser local Storage, section storage or cookies ?

Local storage: Local storage is used to store data in the user's browser indefinitely, while session storage is cleared automatically once the user closes the browser or tab.


Session storage: Session storage is a type of web storage that allows you to store data in the user's browser for the duration of a session, which ends when the user closes the browser or tab.


Cookies: Cookies are small text files that are stored on the user's computer and can be used to store data that needs to persist across sessions..
